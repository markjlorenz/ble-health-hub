## Capturing a session (Wireshark + nRF Sniffer)

1. In Wireshark, under View > Interface Toolbars enable the `nRF Sniffer for Bluetooth LE` option.
2. Set the `Key` dropdown to `Follow LE address`.
3. Set the `Value` to `e4:33:bb:84:83:66 public`.

If you capture a full connection + measurement session, it can be added under `data/` for analysis.
